# OpenAdapter API Skill

A Claude Code / AI agent skill that provides the OpenAdapter API reference and integration guide. One skill gives your agent full knowledge of every OpenAdapter endpoint, SDK setup, and error handling pattern.

## What it does

- Chat completions (OpenAI & Anthropic formats)
- Streaming, tool calling, JSON mode
- Embeddings, image generation, audio transcription, text-to-speech
- Web tools (search, scrape, extract, crawl)
- SDK setup for Python, JavaScript, Go, PHP, Rust, Ruby
- Claude Code / Claude Desktop configuration
- Rate limits and error handling

## Install

```bash
npx skills add k1sh0r/openadapter-api-skill
```

## About OpenAdapter

OpenAdapter is a drop-in replacement for the OpenAI API. One API key gives access to 40+ models from multiple providers. Point any OpenAI-compatible client at the gateway and it just works.

- Base URL: `https://api.openadapter.in`
- Docs: [openadapter.in](https://openadapter.in)
